# 💈 Jeeva Beauty Saloon — React Vite App

## Quick Start
```bash
npm install
npm run dev
# Open http://localhost:5173
```

## Pages
- `/` — Home (Hero, Stats, Services, Barbers, Testimonials)
- `/services` — All services with category filter + cart
- `/booking` — 5-step booking wizard
- `/gallery` — Photo grid + lightbox
- `/contact` — Hours, map, reviews
- `/admin` — Admin dashboard (password: **jeeva2025**)

## Tech Stack
Vite + React 18 · Tailwind CSS · React Router v6 · Framer Motion · React DatePicker · React Hot Toast

## Customize
- **Services & prices** → `src/data/services.js`
- **Barbers** → `src/data/barbers.js`
- **WhatsApp number** → Search `919876543210` in codebase
- **Admin password** → `ADMIN_PASSWORD` in `src/pages/Admin.jsx`

## Build
```bash
npm run build   # Output in /dist
```
